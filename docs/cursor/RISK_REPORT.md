# Cursor Risk Report

Cursor sadece risk durumunda bu dosyayı günceller. Risk yoksa aşağıdaki durum kalır.

```text
Status: clear
Phase: none
Commit: none
Risk: none
Impact: none
Decision needed: no
```

## Kural

- Risk varsa kod yazmayı bırak.
- En fazla 6 satır yaz.
- Devam için kullanıcı onayı bekle.
