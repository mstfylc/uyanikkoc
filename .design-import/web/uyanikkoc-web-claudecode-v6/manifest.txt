UYANIK KOC — WEB UYGULAMASI · CLAUDE CODE HANDOFF (kanonik kaynak)
====================================================================
Bu paket, web uygulamasinin GUNCEL ve TEK kaynagidir. Eski surumler/paketler
DAHIL DEGILDIR. Amac: Claude Code'un bu tasarimi gercek koda BIREBIR uygulamasi.

Tum dosyalar gercek/makine-okur kaynak (text, UTF-8). Path'lerde bosluk/Turkce karakter yok (ASCII).
DIKKAT: Bu ASCII kurali sadece DOSYA ADLARI icindir. ARAYUZ METINLERI TAM TURKCE'dir (ç ğ ı İ ö ş ü)
ve dogruluk kaynagi .jsx dosyalaridir. Kodlarken Turkce karakterleri ASCII'ye dusurme (bkz. CLAUDE.md).
